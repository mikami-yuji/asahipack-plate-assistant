# 落版連絡フォルダ自動生成ツール

システムから毎月出力される「落版候補リスト」のExcelファイルを読み込み、得意先ごとに確認用のExcelファイル（落版確認書）を自動生成し、さらに共有画像フォルダ（`\\asahipack01\画像`）から「受注No.」に対応する画像を自動収集して、顧客別のフォルダに仕分け・保存するNext.jsアプリケーションです。

また、各得意先への連絡用メール下書きを自動生成し、ワンクリックでコピーできる機能も提供します。

---

## 主な機能

1. **落版リスト解析**: アップロードされたシステム生成のExcelファイルをブラウザ上で解析し、得意先ごとに仕分けます。
2. **得意先別フォルダの自動生成**: デスクトップ上に「落版連絡_作成日時」というフォルダを作成し、その中に各得意先専用のフォルダ（例: `0027099_（株）みどりフーズ様`）を自動生成します。
3. **顧客用落版確認Excelの生成**: 得意先用フォルダ内に、指定項目（受注No, Kg, 仕上, 店名, 銘柄, 色数, 色, 最終使用日, 落版予定日）に絞り、回答用のドロップダウン（廃棄/継続）を追加したExcelファイルを自動配置します。※粗利額などの社外秘情報は自動的に除外されます。
4. **対象画像の自動スキャン・コピー**: 共有フォルダ `\\asahipack01\画像` から受注No.で前方一致するデザイン画像を検索し、対象顧客のフォルダ内にコピーします。
5. **メール下書き自動作成**: 顧客名や版件数が埋め込まれた連絡用のメール文章（件名・本文）を表示し、コピーが可能です。

## セットアップ手順

本ツールはローカルPC上で動作するWebアプリケーションです。実行には [Node.js](https://nodejs.org/) (v18以降を推奨) が必要です。

### 1. 依存関係のインストール
プロジェクトのルートディレクトリで以下のコマンドを実行し、必要なライブラリをインストールします。

```bash
npm install
```

### 2. 開発サーバーの起動
以下のコマンドでローカル開発サーバーを起動します。

```bash
npm run dev
```

サーバーが起動したら、ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスします。

### 3. アプリケーションのビルド (本番用)
本番用の最適化ビルドを行う場合は、以下のコマンドを実行します。

```bash
npm run build
npm run start
```

## ユニットテストの実行

主要なロジック（Excel生成、メールテキスト生成など）に対して、JestおよびReact Testing Libraryを用いたユニットテストを作成しています。以下のコマンドでテストを実行できます。

```bash
npm run test
```

---

## 動作要件

* **OS**: Windows (共有フォルダのUNCパスやデスクトップパスの解決にWindows環境を想定しています)
* **実行環境**: Node.js (v18以上), npm (v9以上)
* **ネットワーク**: `\\asahipack01\画像` にアクセス可能な社内ネットワーク環境 (画像コピー機能を使用する場合のみ。アクセスできない場合でもExcelの生成処理は自動で続行されます)
