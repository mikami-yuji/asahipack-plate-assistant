# CHANGELOG

## [1.0.0] - 2026-06-11

### Added
- プロジェクトの初期化 (Next.js App Router, TypeScript, Vanilla CSS)
- `xlsx` および `exceljs` を用いた「落版候補リスト.xlsx」の解析処理を追加
- 顧客用落版確認Excelの生成
  - 指定された表示項目（受注No., Kg, 仕上, 種別, 店名, 銘柄, 色数, 色, 最終使用日, 落版予定日）への絞り込み
  - 仕入先列に基づく「種別」列（シルク印刷/SP/オフセット/カット/アサヒパック/別注）の自動判定と追加
  - 元ExcelのB2セルから担当者名（例: 見上、佐藤など）を自動抽出し、Excelの署名（K3セル）や連絡メールの下書き署名に動的反映する機能を追加
  - 回答欄のドロップダウン（廃棄/継続）および継続理由の入力欄の追加
  - 粗利額などの社内情報の除外
- 共有画像フォルダ (`\\asahipack01\画像`) から受注No.に一致する画像の自動検索とコピー機能を追加
- デスクトップ上への得意先別仕分けフォルダの自動作成処理を実装
- 各得意先ごとの連絡メール下書き（件名・本文）の自動作成とクリップボードコピー機能を実装
- Excel生成ロジック (`excelUtils.test.ts`) およびメール生成ロジック (`emailUtils.test.ts`) のユニットテストを追加
- セットアップ手順などを記載した `README.md` の作成
